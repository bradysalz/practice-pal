### front-end commands

For all of these, make sure to `cd front-end`

-   Start the server: `npm start`
-   Generate `types/supabase.ts`: `npm run gen:types`
-   Generate mock data: `npm run gen:data`
-   Run PR checks:
    -   `npm lint`
    -   `npm typecheck`
    -   `npm test`
